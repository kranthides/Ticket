##Ticketing System

    Project is designed to implement the following functionalities 
    1. Get Latest Seat Count 
    2. Find and Hold best available seats 
    3. Reserve the seats
    
##Design & Architecture 
	
	This project is using following techologies 
		Java 7
		Hibernate
		H2 (In memory database)  
		Maven build tool.
